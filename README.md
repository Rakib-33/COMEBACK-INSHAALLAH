# COMEBACK-INSHAALLAH
How to run the app:
  - First give the student id and password of uniplex and click secure sync that will fetch your incourse all marks for uniplex. (Don't worry your id and password are locally saved inside your phone)
  - Inside the app you will be able to view your total in course mark.
  - Inside each theory course, you will be able to set target grade and the app will show you how much mark you need in final to achieve that grade
  - For sessional, you may give any predicted gpa 
  - Finally click View Gpa to see what gpa you will get by our setted target grades
